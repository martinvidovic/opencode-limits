# Support

opencode-limits is maintained on a best-effort basis without response-time or
resolution-time guarantees.

Support covers the latest stable release and the active prerelease within the
documented OpenCode compatibility range. It covers Provider Adapters bundled
with opencode-limits. Fixes for older releases are provided only at the
maintainer's discretion.

Forks, private modifications, and Provider Adapters not shipped by this
project are outside the support scope.

Use [GitHub Issues](https://github.com/martinvidovic/opencode-limits/issues) to
report bugs or request support. Search existing issues before opening a new
one and include the opencode-limits version, OpenCode version, operating
system, affected provider, reproduction steps, and sanitized output when
available.

Never include access tokens, refresh tokens, cookies, authorization headers,
raw credential files, or other secrets in an issue.
